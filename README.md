# Weighted Flow Matching and Physics-Informed Nonlinear Filtering for Parameter Estimation in Digital Twins

This repository contains a sequential pipeline for learning sample weights, generating reliable inertia samples, and running multiple filtering approaches for spacecraft inertia estimation. The workflow is organized so that all scripts are executed in order through a single main file, while results and figures are saved separately for each stage. 

## Pipeline Overview

The pipeline consists of the following steps:

1. **LRW (`lrw.py`)**  
   Learns sample weights from synthetic and measured angular velocity trajectories and saves the weighted training datasets for different noise levels. 

2. **WFM (`wfm.py`)**  
   Uses the weighted datasets generated by LRW to perform Weighted Flow Matching and generate reliable inertia samples. These generated samples are then used in the filtering stage.

3. **EKF (`ekf.py`)**  
   Runs the Extended Kalman Filter for inertia estimation.

4. **UKF (`ukf.py`)**  
   Runs the Unscented Kalman Filter for inertia estimation.

5. **EnKF (`enkf.py`)**  
   Runs the Ensemble Kalman Filter for inertia estimation.

6. **Boosted UKF (`boosted_ukf.py`)**  
   Uses the generated results from the WFM stage to improve the filtering performance.
